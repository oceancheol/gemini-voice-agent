# Gemini Voice Agent — PRD

## 개요

맥미니에서 실행되는 실시간 음성 대화 웹앱.
아이폰 브라우저로 접속해서 AI 에이전트(원영, 지수, 로즈, 리사)와 전화통화처럼 대화한다.

---

## 배경 및 목표

- 텔레그램은 봇 음성 통화 불가 → 별도 웹앱 필요
- Gemini Live API가 실시간 음성 스트리밍 지원
- 맥미니(항상 켜져 있음) + Tailscale(원격 접속) 환경 활용
- 아이폰 Safari에서 전화 앱처럼 간단하게 사용

---

## 사용자 시나리오

1. 아이폰에서 브라우저 열기
2. 맥미니 주소 접속 (예: `http://macmini.local:3000` 또는 Tailscale 주소)
3. 대화할 에이전트 선택 (원영 / 지수 / 로즈 / 리사)
4. 통화 시작 버튼 → 마이크 권한 허용
5. 말하면 → 에이전트가 실시간으로 음성 응답
6. 통화 종료 버튼

---

## 기능 요구사항

### 필수 기능 (MVP)

| 기능 | 설명 |
|------|------|
| 에이전트 선택 | 원영, 지수, 로즈, 리사 중 선택 |
| 실시간 음성 입력 | 브라우저 마이크 → Gemini Live API 스트리밍 |
| 실시간 음성 출력 | Gemini 응답 → 브라우저 스피커 출력 |
| 통화 시작/종료 | 심플한 UI 버튼 |
| 한국어 지원 | 에이전트별 SOUL.md 성격 반영 |

### 추가 기능 (Phase 2)

- 대화 내용 텍스트 표시 (자막)
- 대화 히스토리 저장
- Tailscale 인증 연동
- 화면 공유 / 카메라 (Gemini 영상 이해)

## 기술 스택

| 구분 | 기술 |
|------|------|
| 서버 | Node.js (Express 또는 Fastify) |
| 실시간 통신 | WebSocket (Gemini Live API ↔ 서버) |
| 프론트엔드 | HTML/CSS/JS (바닐라, 모바일 최적화) |
| 음성 처리 | Web Audio API (브라우저 마이크/스피커) |
| AI | Gemini Live API (`gemini-live` 모델) |
| 실행 환경 | 맥미니 (Node.js 서버 상시 실행) |
| 원격 접속 | Tailscale VPN |

---

## 아키텍처

```
아이폰 브라우저
    │ WebSocket (오디오 스트림)
    ▼
맥미니 Node.js 서버
    │ Gemini Live API WebSocket
    ▼
Google Gemini Live
    │ 음성 응답 스트림
    ▼
맥미니 서버 → 아이폰 브라우저
```

## 에이전트 설정

각 에이전트는 SOUL.md 기반 시스템 프롬프트로 개성 반영:

| 에이전트 | 성격 | 전문분야 |
|---------|------|---------|
| 원영 | 다정한 반말, 친한 친구 톤 | 일반 보조 |
| 지수 | 차분한 반말, 편집자 톤 | 블로그 글쓰기 |
| 로즈 | 활기찬 반말, 에너지 있는 톤 | 숏폼 콘텐츠 |
| 리사 | 간결한 반말, 기술적 톤 | 코딩/자동화 |
